## ğŸ“ Customization

### Adding Images

1. Add your images to the `assets/images/` folder
2. Replace placeholder text `[Image Description]` with actual `<img>` tags
3. Optimize images for web (recommended: WebP format, compressed)

### Updating Content

- School name and branding: Search and replace "Kautilya Educational Academy"
- Contact information: Update in footer sections and contact page
- News and events: Edit the news.html page

### Adding PDFs

1. Place PDF files in `assets/pdf/`
2. Update download links in relevant pages
